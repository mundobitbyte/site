Projeto — Irrigação Automática

Objetivo: compreender como sensores e Arduino podem automatizar tarefas.
1. Introdução Contexto
Arduino • Sensor • IoT

Muitas tarefas do dia a dia podem ser automatizadas utilizando sensores e sistemas eletrônicos.

Neste estudo de caso, você irá analisar um sistema simples de irrigação automática utilizando Arduino.

Sistemas embarcados são sistemas eletrônicos criados para executar funções específicas.
2. Situação-Problema Estudo de caso

Uma pequena plantação está sofrendo porque o solo fica seco em alguns horários do dia.

Para resolver o problema, foi criado um sistema automático de irrigação.

Sensor Detecta a umidade do solo.
Arduino Recebe informações e toma decisões.
Bomba d'água Molha as plantas automaticamente.
O objetivo do sistema é evitar que as plantas fiquem sem água.
3. Conceitos Fundamentais Revisão
Sensor Componente que detecta informações.
Atuador Componente que executa ações.
Microcontrolador Controla o funcionamento do sistema.
Automação Execução automática de tarefas.
IoT Objetos conectados utilizando tecnologia.
Entrada e saída Informações que entram e ações que saem do sistema.
4. Componentes Hardware

O Arduino recebe os dados do sensor e decide quando ligar ou desligar a bomba d'água.

O sensor verifica se o solo está seco ou úmido.

A bomba envia água para as plantas quando necessário.

Sensor → Arduino → Bomba d'água
5. Funcionamento Fluxo
Sensor de Umidade
↓
Arduino
↓
Bomba d'água
Quando o solo está seco, o Arduino liga automaticamente a bomba.
